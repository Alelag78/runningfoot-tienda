RUNNING FOOT Sportswear es un proyecto de tienda online de zapatillas deportivas desarrollado en PHP y MySQL con fines educativos para el módulo Desarrollo Web en Entorno Servidor del ciclo formativo de Técnico Superior en Desarrollo de Aplicaciones Web. El objetivo principal del proyecto es la creación de un e-commerce funcional que permita gestionar productos, usuarios, pedidos y un sistema de stock realista basado en tallas, aplicando los conocimientos adquiridos sobre programación en entorno servidor y bases de datos.

La aplicación permite la gestión de usuarios mediante un sistema de autenticación con registro e inicio de sesión, utilizando sesiones PHP para el control de acceso. Se han definido dos roles diferenciados, administrador y cliente, lo que permite restringir el acceso a determinadas funcionalidades según el tipo de usuario. El administrador tiene acceso completo al sistema, incluyendo la gestión de productos, usuarios y pedidos, mientras que el cliente puede navegar por el catálogo, añadir productos al carrito y realizar pedidos.

El sistema de productos incluye un catálogo público donde se muestran las zapatillas disponibles, así como una página de detalle de producto en la que se presenta información completa y se permite seleccionar talla y cantidad. Uno de los aspectos más relevantes del proyecto es la implementación de un sistema de stock por talla, mediante el cual cada producto puede disponer de diferentes tallas con cantidades independientes. El stock total del producto no se introduce manualmente, sino que se calcula automáticamente como la suma del stock de todas sus tallas, ofreciendo una solución más realista y escalable.

La aplicación incorpora un carrito de la compra gestionado mediante sesiones, que permite añadir productos diferenciando correctamente por talla, modificar cantidades, eliminar artículos y calcular el total del pedido. El proceso de compra incluye una fase de revisión y confirmación, tras la cual se genera un pedido en la base de datos asociando los productos, las tallas seleccionadas, las cantidades y los precios unitarios.

Desde el punto de vista técnico, el proyecto sigue una arquitectura organizada en tres capas, separando las vistas, la lógica de negocio y el acceso a datos. Esta estructura facilita la comprensión del código, su mantenimiento y la posibilidad de futuras ampliaciones. La lógica de negocio se implementa mediante controladores específicos para la autenticación, el catálogo, los productos y los pedidos, mientras que el acceso a la base de datos se centraliza en modelos que encapsulan las consultas SQL.

Para el desarrollo se han utilizado tecnologías como PHP, MySQL, HTML y CSS, empleando XAMPP como entorno de desarrollo local y Visual Studio Code como editor de código. El control de versiones se ha realizado mediante Git y GitHub, permitiendo almacenar y compartir el código fuente del proyecto.

Este proyecto ha sido desarrollado exclusivamente con fines académicos y no está destinado a uso comercial. Las imágenes de productos utilizadas pertenecen a marcas registradas y se han empleado únicamente con propósitos educativos y demostrativos.
